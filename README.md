
# Cambios de Helados — PWA (sin dependencias)

App de una sola página para registrar cambios de helados por canal y motivo.

## Funciones
- Canal: Canal Detalle / Super 24
- Motivo: 50% (energía) o 100% (congelador)
- Búsqueda, scroll, badge con cantidad por producto
- Resumen con `+ / - / Quitar`, Subtotal y **Monto real a aplicar**
- PWA: instalable y con caché offline (service worker)
- **Sin dependencias externas** (HTML + CSS + JS puro)

## Uso local
1. Descomprime el ZIP.
2. Abre `index.html` en tu navegador (Chrome/Edge/Safari).
3. Para instalar: usa “Agregar a la pantalla de inicio” (móvil) o el botón de instalar (desktop).
   - Nota: algunos navegadores exigen servir por HTTP. Si no se instala abriendo el archivo, sirve el folder con `python -m http.server`.

## GitHub Pages
1. Crea un repo.
2. Sube todos los archivos.
3. En Settings > Pages, selecciona la rama `main` y root `/`.
4. Abre la URL pública. ¡Listo!
